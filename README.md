# ***qPCRwm***

**Analysis of qPCR data produced by QuantStudio & StepOne software**

A toolkit for loading and analysing qPCR data produced by QuantStudio & StepOne software.


---


## Installation

You can install the development version of `qPCRwm` from [GitHub](https://github.com/) with:

```r
remotes::install_github("benvallin/qPCRwm")
```
